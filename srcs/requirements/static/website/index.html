What is Inception?

Inception is a system administration project from 42 School focused on creating a complete Docker-based infrastructure.

The objective is to understand containerization, network management, persistent storage, security and service isolation.

Every service runs inside its own container, creating a modular and scalable architecture.

Main Goal

Create a secure production-like environment using multiple Docker containers working together.

Docker Architecture

Nginx

HTTPS web server and reverse proxy handling incoming requests.

WordPress

Dynamic website running with PHP-FPM connected to the database.

MariaDB

Database container storing WordPress data using persistent volumes.
